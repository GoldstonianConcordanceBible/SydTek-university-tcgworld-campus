# Zone — Mirror → Water → Fire Hall (Method Mastery)

## Purpose
Teach the interpretive method as a disciplined workflow:
- Mirror: evidence + definitions + boundaries
- Water: synthesis + tensions + hard question
- Fire: application + accountability + guardrails

## What exists here (minimum)
- Mirror Wing prompt board
- Water Wing cross-link board
- Fire Wing application + guardrail board
- Docent NPC placement
- q003 start board

## Player Flow
1) Choose prompt  
2) Write Mirror notes  
3) Build Water synthesis  
4) Write Fire action + accountability  
5) Submit tri-fold memo

## Required Sign Text (short)
**Mirror:** observe + define + boundaries  
**Water:** connect + synthesize + tensions  
**Fire:** apply + account + guardrails

## Quests Routed
- q003_mirror-water-fire

## NPCs Routed
- Docent (primary)
- Librarian (anchors/cross-links support)

## Completion Signal
Learner submits Tri-Fold Method Memo proof artifact.
