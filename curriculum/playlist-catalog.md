# Playlist Catalog

This document tracks core SydTek learning playlists.

Each playlist should eventually include:

• title  
• pathway  
• level  
• estimated hours  
• associated book  
• assessment type  
• registry identifier

## Expansion Rule

Every playlist added to the catalog should be tied to:

• a defined pathway  
• an estimated duration  
• a book or reading requirement  
• at least one reflective or demonstrative assessment
