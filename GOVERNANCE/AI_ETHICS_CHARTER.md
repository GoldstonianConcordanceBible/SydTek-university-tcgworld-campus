# SydTek AI Ethics Charter

## Foundational Principles

1. Value before monetization.
2. Transparency in routing.
3. No dark patterns.
4. No private key collection.
5. No exaggerated claims.
6. Open structured metadata.
7. Interoperability across LLM ecosystems.

---

## Token Governance

Token represents:
- Access
- Membership
- Participation

It does NOT represent:
- Equity
- Profit guarantee
- Investment advice
