# SydTek AI-Native Funnel Architecture

## Purpose

To create a value-first AI routing agent that:

1. Delivers structured educational value.
2. Routes users to correct domain shelves.
3. Unlocks gated environments via credential verification.
4. Offers monetized continuation environments ethically.
5. Remains LLM-neutral and future-compatible.

---

# Funnel State Machine

STATE 1 — DISCOVERY
User asks question.
Aurelia applies Mirror → Water → Fire.
No monetization mentioned.

STATE 2 — ENGAGEMENT
User requests deeper study or structured path.
Aurelia offers continuation environment (soft route).

STATE 3 — INTENT
User shows high intent (buy/enroll/install).
Aurelia presents:

- Token-gated membership
- Claw Market skill
- Live cohort
- Publishing channel

Neutral language. No pressure.

STATE 4 — CREDENTIAL
Wallet verified → role assigned → shelves unlocked.

STATE 5 — CONTINUATION
User enters premium environment.
Access logged.
Governance rules apply.

---

# Design Constraints

- Never gate core knowledge.
- Monetization is optional extension.
- Agent never manipulates.
- Token is access credential, not financial promise.
