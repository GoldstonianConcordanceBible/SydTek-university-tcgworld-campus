# Algebra I — Assessment Pack (SYD-912-MATHA-A1)

This file defines what assessments must exist for a complete course. It does not include copyrighted test banks; it is a SydTek-original specification.

## Minimum Assessments
### Per Unit
- 2–4 formative checks (exit tickets, mini-quizzes, short sets)
- 1 unit summative (quiz/test/performance task)

### Per Quarter (9 weeks)
- 1 cumulative review assessment

### End of Course
- Final Exam (cumulative)
- Final Project (P08) + rubric-based defense

## Summative Structure (recommended)
- Part A: skills (15–20 items)
- Part B: reasoning (2–4 items)
- Part C: modeling (1–2 tasks)

## Common Rubrics
Use `/academy/sydtek-scholars/ASSESSMENT/RUBRICS.md`:
- Math reasoning
- Project/presentation
