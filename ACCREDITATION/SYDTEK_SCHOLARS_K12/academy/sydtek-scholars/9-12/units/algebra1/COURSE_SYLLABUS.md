# Algebra I — Course Syllabus (SYD-912-MATHA-A1)
**Credit:** 1.0  
**Recommended Grade:** 8–10 (placement-based)  
**Prereqs:** integer operations; fraction/decimal/percent fluency; basic equations  
**Formation Spine:** Mirror → Water → Fire  
**Mastery Scale:** See `/academy/sydtek-scholars/ASSESSMENT/MASTERY_SCALE.md`

## Course Outcomes
Learner will:
- simplify expressions using properties
- solve linear equations and inequalities and justify steps
- model and interpret linear functions
- solve systems with multiple methods and interpret results
- apply exponent rules and scientific notation
- factor polynomials and solve quadratics by factoring
- complete an evidence-based portfolio and a final modeling project

## Required Evidence (Portfolio)
See `PORTFOLIO_ARTIFACTS.md` (8 artifacts minimum).

## Grading (suggested)
- Unit Summatives (12): 50%
- Quizzes/Checks: 20%
- Projects/Portfolio: 20%
- Final Exam: 10%

**Pass rule:**  
- Level 3 (Meets) in at least 10/12 unit summatives  
- all portfolio artifacts complete  
- Final Project at rubric Level 3

## Weekly Rhythm (parent-operable)
- 4 instructional days + 1 assessment/review day (recommended)
- Daily: warm-up (5–10m), instruction (10–15m), guided practice (15–20m), independent practice (15–25m), exit ticket (3–5m)

## Integrity / Work Standards
- Show steps for multi-step work.
- Calculator is verification unless a lesson explicitly allows it.

## Retake Policy (mastery-based)
- Identify error types → targeted re-teach → parallel reassessment.
