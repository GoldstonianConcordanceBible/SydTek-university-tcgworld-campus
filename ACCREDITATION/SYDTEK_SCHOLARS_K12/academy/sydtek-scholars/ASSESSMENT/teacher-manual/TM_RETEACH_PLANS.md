# Teacher Manual — Reteach Plans (Algebra I)

## Error-Type Reteach Map
### 1) Sign errors
- Use number line demonstrations
- Require rewriting as “+ (-3)” style before combining

### 2) Distribution errors
- Use area model/box method
- Provide “distribute checklist”

### 3) Two-step equation confusion
- Require inverse-operation arrows
- One operation per line rule

### 4) Inequality flip errors
- Teach with counterexample: test a number to show why flip matters

### 5) Slope confusion
- Require labeling points (x1,y1), (x2,y2)
- Use table-first approach

### Reassessment
- After reteach: 8–12 item parallel check
