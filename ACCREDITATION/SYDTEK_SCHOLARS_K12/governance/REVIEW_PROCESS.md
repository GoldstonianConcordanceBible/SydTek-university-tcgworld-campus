# Review Process

## Unit acceptance checklist
- measurable objectives
- clear lesson flow parent can run
- formative + summative assessment
- accommodations
- Mirror → Water → Fire mapping
- recordkeeping evidence defined
- no copyright reproduction
- correct non-affiliation language if referencing Abeka

## Required approvals
- New units: Tier 2 + Tier 3
- Ops/legal docs: Tier 3
