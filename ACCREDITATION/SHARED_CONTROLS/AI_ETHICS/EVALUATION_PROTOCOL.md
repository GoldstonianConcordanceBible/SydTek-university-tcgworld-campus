# AI Evaluation Protocol

**Owner:** AI Governance Lead  
**Review:** Quarterly

## 1. Risk tiers
- Tier 1 (Low): study aids, summaries
- Tier 2 (Medium): tutoring suggestions, learning pathway recommendations
- Tier 3 (High): grading, placement, discipline, credentialing influence

## 2. Required evaluations by tier
### Tier 1
- Basic safety tests + hallucination checks

### Tier 2
- Add bias evaluation + explainability check + drift monitoring

### Tier 3
- Add formal rubric validation + human review workflow + audit logs + appeal mechanism

## 3. Approval gate
Tier 3 systems require written approval by:
- Academic Director
- AI Governance Lead
- Safeguarding Lead (if minors involved)

## 4. Logging
Logs must support:
- decision trace
- user appeal
- incident investigations
