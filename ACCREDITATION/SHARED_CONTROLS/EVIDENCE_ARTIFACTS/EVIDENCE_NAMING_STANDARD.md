# Evidence Naming Standard

## Goal
Make every artifact easy to locate and tie to policy claims.

## Naming format
YYYY-MM-DD__AREA__TOPIC__VERSION__REDACTED(if needed)

Examples:
- 2026-02-24__K12__BENCHMARK_REPORT_Q1__v1__REDACTED.pdf
- 2026-03-01__AI__BIAS_AUDIT_TIER2__v1.md
- 2026-03-15__SAFEGUARDING__INCIDENT_LOG_SUMMARY_Q1__v1.md

## Redaction rules
- Remove personal identifiers (names, DOB, addresses)
- Use student IDs or placeholders
- Keep timestamps + categories for audit usefulness
