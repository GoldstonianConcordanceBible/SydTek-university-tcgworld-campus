# Course Shell Evidence Checklist

A course is “audit-ready” when it includes:

- [ ] Course description
- [ ] Learning outcomes
- [ ] Weekly/unit plan
- [ ] Assignments with rubrics
- [ ] Grading weights
- [ ] Academic integrity + AI disclosure rules
- [ ] Accessibility accommodations note
- [ ] Privacy/records note
- [ ] Communication expectations
- [ ] Evidence: at least 2 graded samples (redacted)
