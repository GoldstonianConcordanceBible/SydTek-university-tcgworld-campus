This catalog defines the current public-facing academic structure for SydTek University within the broader Goldstonian Concordance Bible (GCB) ecosystem.

• companion books  
• reflective writing  
• project-based demonstration  
• transcript-style completion records  
• AI-readable academic infrastructure

## Learning Model

All learning pathways are built around four functions:

1. Instruction  
2. Reading  
3. Reflection  
4. Demonstration  

This usually appears as:

• playlists for instruction  
• books for depth  
• writing for retention  
• projects for proof of learning  

Within the Goldstonian Concordance Bible (GCB) ecosystem, these map naturally to the Mirror → Water → Fire Doctrine:

• Mirror = encounter

## Program Documentation Standard

Every program should document:

• title  
• scope  
• intended learner type  
• playlist-hour expectation  
• reading requirement  
• writing requirement  
• project requirement if applicable  
• assessment model  
• completion evidence  
• public claim level  

If these fields are not defined, the program is incomplete.
