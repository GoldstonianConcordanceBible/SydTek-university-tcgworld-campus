# SydTek University — Accreditation Readiness Dossier

This folder organizes SydTek University documentation in an accreditor-friendly format:
- mission alignment
- academic governance
- program architecture
- learning outcomes
- assessment and continuous improvement
- faculty qualification documentation
- student support services
- learning resources
- evidence indexing

## How to Navigate
- `/00-institution-overview/` — institutional policies, outcomes, governance, and QA systems
- `/01-schools/` — school-by-school program descriptions with outcomes and assessment plans
- Each school includes an `evidence-index.md` to link artifacts (syllabi, rubrics, minutes, catalogs, etc.)

## Evidence Philosophy
Evidence is referenced as stable links (GitHub paths, PDFs, or external persistent identifiers when available).
All claims should be supportable by evidence entries in each `evidence-index.md`.
