# Program Handbook — Certificate in Biblical Studies Foundations

## Program Overview
- Credential Level: Certificate
- School: Biblical & Theological Studies
- Delivery Mode: Online / MOOC-supported (video + GitHub materials + assessments)
- Typical Length: 3–6 weeks part-time (self-paced)
- Pathways: Prepares learners for further Bible study series (Gospels, Prophets, Epistles, Second Temple literature) and for SydTek verified certificates.

## Required Courses
- BTS-000 Orientation: How SydTek University Works
- BTS-101 The GCB Method: Mirror → Water → Fire Hermeneutics
- BTS-102 Torah Foundations: Covenant, Exodus, Law, Tabernacle

## Admission Requirements
- None (open enrollment)
- Basic reading/writing ability recommended

## Completion Requirements
- Successful completion of required assessments in each required course
- Submission of portfolio artifacts:
  - Learning Covenant (BTS-000)
  - Passage Analysis (BTS-101)
  - Torah Theme Map (BTS-102)
