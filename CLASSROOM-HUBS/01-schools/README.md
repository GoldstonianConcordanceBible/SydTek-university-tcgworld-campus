# Schools Directory

Each school folder contains:
- `README.md` — accreditor-facing overview and scope
- `programs.md` — program list with credential levels and delivery modes
- `learning-outcomes.md` — school-level outcomes measurable across programs
- `curriculum-map-template.md` — course-to-outcomes mapping template
- `assessment-plan.md` — assessment methods, cycle, and improvement process
- `faculty-credentials-template.md` — roles, minimum qualifications, and dossier links
- `resources.md` — facilities, labs, software, library resources
- `evidence-index.md` — artifact links supporting the school’s claims
