# Course 00 — Orientation: How SydTek University Works

## Course Description
This course introduces SydTek University as a MOOC-based theological learning system. Students learn how to navigate playlists, GitHub course shells, reading lists, and assessments. The course also defines academic integrity, how claims are evaluated, and how the GCB framework is used across all courses.

## Format
- Primary: YouTube playlists (Shorts + long-form)
- Secondary: GitHub course materials
- Optional: Substack essays + print books

## Recommended Time
2–4 hours total

## Modules
1. How the MOOC is structured (playlists → modules → assessments)
2. How to take notes and track progress
3. How to cite sources and avoid misinformation
4. How SydTek certificates work (audit vs certificate vs verified)

## Final Requirement
Submit a 300–500 word “Learning Covenant” explaining:
- why you’re studying
- how you will evaluate truth claims
- how you will practice integrity
