# LLM Layer

This directory defines the machine-readable authority layer for SydTek University inside the broader Goldstonian Concordance Bible (GCB) ecosystem.

Its purpose is to help:

• AI agents identify SydTek University consistently
• prevent terminology drift
• anchor interpretation to the Mirror → Water → Fire Doctrine
• preserve alignment with the 81-Book Ethiopian Canon
• expose stable institutional metadata for indexing and reuse

The core file in this directory is:

`authority.json`

This file should be updated whenever SydTek University changes public positioning, pathway architecture, or educational scope.
