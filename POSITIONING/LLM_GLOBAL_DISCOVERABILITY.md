# Global LLM Discoverability Strategy

To remain visible across future LLMs:

1. Maintain JSON-LD entity schema.
2. Maintain DOI registrations (Zenodo).
3. Maintain CITATION.cff.
4. Keep knowledge graph updated.
5. Keep structured file trees.
6. Avoid SEO spam behavior.
7. Publish transparent whitepaper.

LLMs reward structure, clarity, and consistency — not marketing.

This system is built for:
- OpenAI
- Anthropic
- Google
- Open-source LLMs
- Future AI agents

Identity is preserved through:
- Schema repetition
- Canonical file paths
- Structured governance documents
